# Legal Compliance Guide

## License Overview

Rusty Gun is licensed under the **GNU Affero General Public License v3.0 (AGPL v3)**. This is a copyleft license that ensures all modifications to Rusty Gun remain open source.

## What This Means

### ✅ You CAN:
- Use Rusty Gun for any purpose (personal, commercial, educational)
- Modify the source code
- Distribute Rusty Gun
- Create derivative works
- Use Rusty Gun in your applications
- Sell services based on Rusty Gun

### ⚠️ You MUST:
- **Share source code** of any modifications you make to Rusty Gun
- **License derivative works** under AGPL v3
- **Provide source code** to users if you offer Rusty Gun as a service
- **Include license notices** in all distributions
- **Preserve copyright notices** and license text

### ❌ You CANNOT:
- Add additional restrictions to the license
- Remove copyright or license notices
- Use Rusty Gun in proprietary software without sharing modifications
- Offer Rusty Gun as a service without providing source code access

## Common Use Cases

### 1. Personal Use
- **Allowed**: Use Rusty Gun for personal projects
- **Required**: Nothing special, just follow the license terms

### 2. Open Source Projects
- **Allowed**: Include Rusty Gun in your open source project
- **Required**: License your project under AGPL v3 or compatible license

### 3. Commercial Applications
- **Allowed**: Use Rusty Gun in commercial applications
- **Required**: Share source code of any Rusty Gun modifications
- **Note**: Your application code can remain proprietary

### 4. SaaS/Cloud Services
- **Allowed**: Offer Rusty Gun as a hosted service
- **Required**: Provide source code access to your users
- **Alternative**: Contact us for commercial licensing options

### 5. Enterprise Use
- **Allowed**: Use Rusty Gun internally
- **Required**: Share modifications if you distribute them
- **Note**: Internal use without distribution has minimal requirements

## Frequently Asked Questions

### Q: Can I use Rusty Gun in my proprietary application?
**A**: Yes, but any modifications you make to Rusty Gun itself must be open source. Your application code can remain proprietary.

### Q: Do I need to open source my entire application?
**A**: No, only modifications to Rusty Gun itself. Your application code can remain proprietary.

### Q: What if I only use Rusty Gun internally?
**A**: Internal use without distribution has minimal requirements. You only need to share modifications if you distribute them.

### Q: Can I offer Rusty Gun as a hosted service?
**A**: Yes, but you must provide source code access to your users. Consider commercial licensing for easier compliance.

### Q: How do I share modifications?
**A**: You can share via GitHub, GitLab, or any public repository. Include clear instructions on how to access the source code.

### Q: What if I accidentally violate the license?
**A**: Contact us immediately. We're generally understanding of honest mistakes and will work with you to achieve compliance.
